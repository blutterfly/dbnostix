    13 üì° Web Development

    Python is a powerful language for web development, offering a variety of frameworks for building web applications, APIs, and documentation sites. This chapter covers Flask, FastAPI, MkDocs, and Streamlit, widely used for web applications, REST APIs, documentation, and data visualization.

    13.1 üåç Overview of Python Web Development

    Python provides lightweight and scalable web frameworks for different needs:

    Framework Purpose
    Flask Lightweight, flexible web framework
    FastAPI High-performance API framework for modern web applications
    MkDocs Static site generator for documentation
    Streamlit Framework for building data-driven web apps

    13.2 üèóÔ∏è Flask ‚Äì Lightweight Web Framework

    Flask is a minimalistic and easy-to-use web framework, widely used for small to medium-sized web apps.

    ‚úÖ Installing Flask

    pip install flask

    üîπ Creating a Simple Flask App

    from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "Hello, Flask!"

if __name__ == "__main__":
    app.run(debug=True)

    üîπ Run the app:

    python app.py

    üîπ Visit http://137.0.0.1:5000/ in your browser.

    üîπ Handling Routes & URL Parameters

    @app.route("/user/<name>")
def greet_user(name):
    return f"Hello, {name}!"

    ‚úÖ Use Case: Building web dashboards, small applications, and APIs.

    13.3 üöÄ FastAPI ‚Äì High-Performance APIs

    FastAPI is a modern, fast framework for building APIs with automatic documentation and async support.

    ‚úÖ Installing FastAPI

    pip install fastapi uvicorn

    üîπ Creating a Simple FastAPI App

    from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def home():
    return {"message": "Hello, FastAPI!"}

if __name__ == "__main__":
    import uvicorn
    uvicorn.run(app, host="137.0.0.1", port=8000)

    üîπ Run the app:

    uvicorn app:app --reload

    üîπ Visit the automatic API docs:

    • Swagger UI: http://137.0.0.1:8000/docs
    • ReDoc: http://137.0.0.1:8000/redoc

    üîπ Handling API Requests

    @app.get("/items/{item_id}")
def read_item(item_id: int):
    return {"item_id": item_id}

    ‚úÖ Use Case: Building REST APIs, microservices, AI model deployment.

    13.4 üìñ MkDocs ‚Äì Documentation for Python Projects

    MkDocs is a static site generator used for creating project documentation.

    ‚úÖ Installing MkDocs

    pip install mkdocs

    üîπ Creating a New Documentation Site

    mkdocs new my_docs
cd my_docs
mkdocs serve

    üîπ View your documentation:
    Visit http://137.0.0.1:8000/ in your browser.

    üîπ Adding Markdown Content

    Edit docs/index.md:

    # Welcome to My Docs
This is a test documentation page.

    üîπ Building the Site

    mkdocs build

    ‚úÖ Use Case: Project documentation, API documentation, internal guides.

    13.5 üìä Streamlit ‚Äì Building Data-Driven Web Apps

    Streamlit is a simple framework for building interactive web applications for data visualization.

    ‚úÖ Installing Streamlit

    pip install streamlit

    üîπ Creating a Simple Streamlit App

    Create app.py:

    import streamlit as st

st.title("Hello, Streamlit!")
st.write("This is a simple web app using Streamlit.")

    üîπ Run the app:

    streamlit run app.py

    üîπ View the app in the browser at http://localhost:8501/.

    üîπ Adding User Input and Charts

    import pandas as pd
import numpy as np

data = pd.DataFrame(
    np.random.randn(10, 2),
    columns=['A', 'B']
)

st.line_chart(data)

    ‚úÖ Use Case: Building dashboards, AI model visualizations, interactive reports.

    üöÄ Summary

    Framework Purpose Best For
    Flask Simple web applications Small apps, dashboards
    FastAPI High-speed APIs REST APIs, ML model deployment
    MkDocs Documentation generator API and project documentation
    Streamlit Data-driven web apps ML dashboards, data visualization
