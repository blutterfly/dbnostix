    11 üåê File Handling

    File handling is an essential part of programming, allowing reading, writing, and manipulating files. In data science, machine learning, and AI, working with text files, CSV files, and logs is common for storing and processing large datasets.

    This chapter covers file handling operations, including reading, writing, appending, working with different file formats, and error handling.

    11.1 üìÇ Working with Files in Python

    Python provides built-in functions for opening, reading, writing, and closing files.

    ‚úÖ Opening a File

    file = open("example.txt", "r")  # Open a file in read mode
print(file.read())  # Read the entire file content
file.close()  # Always close the file after use

    ‚úÖ Use Case: Reading log files, dataset files, configuration files.

    11.2 üìù Writing to a File

    To write to a file, use write ('w') mode. If the file doesn‚Äôt exist, Python creates a new file.

    ‚úÖ Writing Data

    file = open("example.txt", "w")  # Open in write mode
file.write("Hello, Python!\nWelcome to File Handling.")
file.close()

    üîπ Note: 'w' mode overwrites existing content. To append, use 'a'.

    ‚úÖ Appending Data to an Existing File

    file = open("example.txt", "a")
file.write("\nThis is an additional line.")
file.close()

    ‚úÖ Use Case: Logging data, saving AI/ML model outputs.

    11.3 üìñ Reading Files Efficiently

    Python provides different ways to read files efficiently.

    ‚úÖ Reading a File Line by Line

    file = open("example.txt", "r")

for line in file:
    print(line.strip())  # Removing extra newlines
file.close()

    ‚úÖ Use Case: Processing large datasets without memory overflow.

    üîπ Using readlines() to Read All Lines as a List

    file = open("example.txt", "r")
lines = file.readlines()
print(lines)  # Output: List of lines in the file
file.close()

    ‚úÖ Use Case: Reading structured text data (e.g., logs, reports).

    11.4 üöÄ Using with Statement for File Handling

    Using with open() ensures that the file automatically closes after execution.

    ‚úÖ Safe File Handling with with

    with open("example.txt", "r") as file:
    content = file.read()
    print(content)

    ‚úÖ Use Case: Better memory management, avoiding file lock issues.

    11.5 üìä Handling Different File Formats

    üîπ Working with CSV Files (csv Module)

    import csv

with open("data.csv", "r") as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)  # Output: List of values in each row

    ‚úÖ Use Case: Reading structured datasets in ML models.

    üîπ Writing to a CSV File

    with open("output.csv", "w", newline="") as file:
    writer = csv.writer(file)
    writer.writerow(["Name", "Age"])
    writer.writerow(["Alice", 25])

    ‚úÖ Use Case: Saving processed data from AI pipelines.

    üîπ Working with JSON Files (json Module)

    import json

data = {"name": "Alice", "age": 25, "city": "New York"}

# Writing JSON data
with open("data.json", "w") as file:
    json.dump(data, file)

# Reading JSON data
with open("data.json", "r") as file:
    loaded_data = json.load(file)
print(loaded_data)  # Output: {'name': 'Alice', 'age': 25, 'city': 'New York'}

    ‚úÖ Use Case: Handling API responses, configuration files, AI model metadata.

    11.6 üõ†Ô∏è Error Handling in File Operations

    It‚Äôs important to handle file-related errors to prevent crashes.

    ‚úÖ Handling File Not Found Error

    try:
    with open("non_existent.txt", "r") as file:
        content = file.read()
except FileNotFoundError:
    print("Error: File not found!")

    ‚úÖ Use Case: Ensuring robust scripts in production AI pipelines.

    11.7 üìÇ Working with Directories (os Module)

    The os module allows working with directories and file management.

    ‚úÖ Listing Files in a Directory

    import os

print(os.listdir("."))  # Lists all files in the current directory

    üîπ Creating and Deleting Folders

    os.mkdir("new_folder")  # Create a folder
os.rmdir("new_folder")  # Remove a folder

    ‚úÖ Use Case: Managing dataset directories in ML projects.
