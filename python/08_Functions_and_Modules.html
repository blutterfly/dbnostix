    08 üé≠ Functions and Modules

    Functions and modules are essential building blocks in Python, promoting code reuse, modularity, and maintainability. Understanding them is crucial for data science, machine learning, and AI, where reusable code improves efficiency and readability.

    This chapter covers defining functions, argument handling, lambda functions, recursion, and working with modules to write efficient, modular, and scalable Python code.

    08.1 üéØ Functions: The Building Blocks of Python

    A function is a reusable block of code that performs a specific task.

    ‚úÖ Defining a Function

    def greet(name):
    return f"Hello, {name}!"

print(greet("Alice"))  # Output: Hello, Alice!

    ‚úÖ Use Case: Encapsulating repetitive code, making programs more readable.

    08.2 üé≠ Function Arguments and Parameters

    Functions in Python support different types of arguments:

    üîπ Positional Arguments

    def add(a, b):
    return a + b

print(add(5, 3))  # Output: 8

    üîπ Default Arguments

    def power(base, exponent=2):
    return base  exponent

print(power(3))     # Output: 9 (3¬≤)
print(power(3, 3))  # Output: 27 (3¬≥)

    üîπ Keyword Arguments

    print(power(exponent=3, base=2))  # Output: 8

    üîπ Variable-Length Arguments (*args, **kwargs)

    def sum_all(*numbers):
    return sum(numbers)

print(sum_all(1, 2, 3, 4))  # Output: 10

def display_info(info):
    print(info)

display_info(name="Alice", age=25)

    ‚úÖ Use Case: Handling dynamic data inputs in ML models, APIs, and automation scripts.

    08.3 ‚ö° Lambda (Anonymous) Functions

    Lambda functions are short, one-line functions often used in data processing.

    ‚úÖ Lambda Syntax

    square = lambda x: x  2
print(square(5))  # Output: 25

    üîπ Lambda with map(), filter(), reduce()

    numbers = [1, 2, 3, 4, 5]
squared = list(map(lambda x: x2, numbers))
print(squared)  # Output: [1, 4, 9, 16, 25]

    ‚úÖ Use Case: Data transformations in pandas, NumPy, and machine learning preprocessing.

    08.4 üîÅ Recursion: Functions Calling Themselves

    Recursion is used when a problem can be broken down into smaller subproblems.

    ‚úÖ Factorial Calculation

    def factorial(n):
    if n == 1:
        return 1
    return n * factorial(n - 1)

print(factorial(5))  # Output: 120

    ‚úÖ Use Case: Tree-based algorithms, graph traversal (DFS), Fibonacci sequence.

    08.5 üì¶ Python Modules (Importing and Organizing Code)

    Modules allow code organization by grouping related functions and variables.

    ‚úÖ Importing a Module

    import math
print(math.sqrt(25))  # Output: 5.0

    üîπ Importing Specific Functions

    from math import sqrt
print(sqrt(16))  # Output: 4.0

    üîπ Creating a Custom Module

    üìå Create a file mymodule.py

    üìå Import and Use the Module

    ‚úÖ Use Case: Reusing functions in large AI projects, ML models, and APIs.
